# Dicas na aula

Quando você vai printar

> print('You're in a cat's land. Choose the colour of the cat "black" or "white')

Ele vai dar erro, pois vai pensar que as partes abaixo são blocos

>You

> s land. Choose the colour of the cat "black" or "white

Como corrigir? Com \

print('You\'re in a cat\'s land. Choose the colour of the cat "black" or "white')

Assim ele sabe que o ' no meio, é um texto
